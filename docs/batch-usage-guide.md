# 批量发布使用指南

## 概览

51guapi 支持三档发布模式，满足不同安全等级的发布需求：

| 档位         | 图标 | 行为                    | 适用场景                           |
| ------------ | ---- | ----------------------- | ---------------------------------- |
| `off`        | ⏻    | 只填充不发布            | 日常编写、人工完整审核后手动发布   |
| `dry-run`    | 🧪   | 预演流程(填充+发布模拟) | 验证生成质量、表单兼容性、正文渲染 |
| `authorized` | 🔴   | 真实发布                | 确认无误后进入正式发布模式         |

## 界面导航

1. 在 Chrome 扩展工具栏点击图标打开 Side Panel
2. 点击 **≣ 批量** 进入批量视图
3. 左上标签页可在 **批次** 和 **历史** 之间切换

## 档位说明

### off 档位（默认）

- 点击「填充到当前页」仅填充表单
- 需要人工手动点击后台的「发布」按钮
- 安全默认，首次使用或任何时候都可用

### dry-run 档位

- 点击「预演」后：
  - 生成草稿 → 填充表单 → 执行发布动作（实际不提交）
  - DryRunReport 展示填充结果（✓已填 / ↷跳过 / ⚠降级）
- 用于验证整个流程是否正常工作
- 可随时切换回 `off` 或进入 `authorized`

### authorized 档位

- **红色警告**：任意批准操作都会真实发布到后台
- 需输入 `publish` 手势确认（防误触）
- 点击「批准发布」后立即生效

## 使用流程

### 1. 输入选题并生成草稿

```
在批量视图输入选题(每行一条):
某新番看点
某里番作品介绍
```

点击「开始批量(生成+填充)」，系统依次生成草稿并存入批次。

### 2. 审核草稿

批次创建后自动进入 **待审** 状态：

- 点击每个选题可展开查看/编辑草稿
- 正文字段唯读，其他字段可编辑
- 可随时点击 **急停** 中止整个批次

### 3. 切换档位并批准

#### dry-run 预演

1. 在 Settings 页设置档位为 `dry-run`
2. 返回批量视图，点击「预演 N 条」
3. 查看 DryRunReport 确认填充结果
4. 如有问题，人工编辑草稿后重试

#### authorized 真发

1. 在 Settings 页设置档位为 `authorized`
2. 返回批量视图，点击「批准发布 N 条」
3. 输入 `publish` 确认手势
4. 点击「确认」执行真实发布

### 4. 漂移自检（重要）

Before approving, the system performs an automatic selector drift check. If the admin page has been modified (layui form structure changes), the check may fail with missing selectors.

- **绿色通过**：继续批准流程
- **橙色警告**：点击「跳过检查继续批准」或刷新后台页后重试

## 隔离仓处理流程 (needs-human-verification)

### 什么时候会出现

- 发布过程中 Service Worker 被回收
- 标签页在填充前关闭
- content script 执行异常

### 正确处理方式

1. **看到橙色警告**：`⚠ N 条需人工核对`
2. **进入隔离态**：点击「我已核对,撤出隔离」需满足：
   - 若显示「查看帖子」链接，先点击确认后台已发布
   - 确定帖子存在后再撤出隔离
3. **重试**：无链接时可直接重试该条目

> 系统绝不会自动重发隔离条目，防止重复发布。

## 填充结果三态说明

| 状态     | 图标 | 含义                               | 处理          |
| -------- | ---- | ---------------------------------- | ------------- |
| filled   | ✓    | 字段已成功填充                     | 无需处理      |
| skipped  | ↷    | 选择器未找到（表单变更）           | 刷新页后重试  |
| degraded | ⚠    | innerHTML 降级填充（格式可能丢失） | 手动检查/粘贴 |

---

## 历史记录

- 点击 **历史** 标签页查看发布轨迹
- 每条记录包含选题、标题、状态、时间
- `publish-confirmed` 记录可点击「查看帖子」跳转后台

---

## 崩溃恢复

- Side Panel 刷新/重启后自动恢复活跃批次状态
- 标签页关闭后重新打开，tombstone 协议处理飞行中条目
- 发布中断条目自动进入隔离，需人工核对
