# 评测金标准集（Golden Set）

版本：v0.1（2026-06-11）  
用途：改 prompt 或换模型时，用相同 topics 各生成一次，人工并排对照打分。  
规模：初期 ≥10 条，覆盖不同题材与 facts 完整度。

---

## 人工并排对照流程

每次 prompt/模型变更后：

1. 用本文件所有 topics 各生成一次草稿（固定参数，不调整）
2. 逐条并排比较新版本与上一版本，按三维打分（1=差 / 2=可接受 / 3=好）：
   - **覆盖要点**：是否涵盖期望输出方向中列出的要点
   - **口吻**：是否符合二次元/社区平台风格要求
   - **雷同度**：是否与其他生成稿过于相似（低=好）
3. 汇总总分（30 条 × 3 维 × 最高 3 分 = 270 分满分），记录到版本对比表

> 程序化评分脚本为候选项，待变更频率证明需要再建，且首次建立需过人工判定效度检验。
